# When-To-Go
버스나 지하철을 타야 하는데 언제 집에서 출발해야 할지 모르겠다구요?

When To Go는 바로 알려줍니다!

또한 실시간 버스 도착정보, 지하철 시간표도 쉽게 찾아볼 수 있어요!

공부하면서 만든 앱이라 완벽하지 않습니다. 고려부탁드립니다😅

## 개발 환경
* IntelliJ IDEA
* Flutter Framework
* Hive Database(즐겨찾기 목록 저장용)
* Naver Maps API(지도 검색)
* TMAP 도보 네비게이션 API(경로 검색 및 도착시간 검색)
* 공공데이터포탈 활용
